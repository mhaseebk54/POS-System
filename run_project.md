---
description: Run the Laundry POS System (Full Stack)
---
1. Start the Backend Server
   - Open a terminal.
   - Run the following command:
   ```bash
   cd server
   npm start
   ```

2. Start the Frontend Client
   - Open a new terminal.
   - Run the following command:
   ```bash
   cd client
   npm run dev
   ```

3. Access the Application
   - Open your browser and go to `http://localhost:5173` (or the port shown in the client terminal).
   - Log in with credentials:
     - Email: `admin@pos.com`
     - Password: `admin123`
